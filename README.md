# MONO MAZE  
Escape the Infinite Maze

MONO MAZE は、シンプルなモノクロスタイルで構成された  
**無限迷路脱出ゲーム**です。  
ランダム生成される迷路を進み、アイテムを集め、  
出口を目指して脱出してください。

---

## 🎮 Features（特徴）

- **無限に遊べるランダム迷路生成**
- **視界制限による探索感の強化**
- **シード値入力で同じ迷路を再現可能**
- **シンプルで軽量、ブラウザだけで遊べる**

---

## 🕹️ How to Play（遊び方）

### 移動
- **W / ↑** : 上へ  
- **A / ←** : 左へ  
- **S / ↓** : 下へ  
- **D / →** : 右へ  
スマホはタッチで操作可能
